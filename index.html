Welcome

This is a simple stub for the Code Docs Codex. I plan on dressing some bones on these bones someday--and possibly even some meat, too!

Purpose

The Codex is meant to collect my favorite programming documentation, and edit it thoroughly to make it both legible & current. As it is, so much of the documentation I regularly reference is either out of date or written by monkeys.

To Use

I don't have the skill--nor the time to acquire the skill--to make a decent website. I'm also cheap, or else I would use Striking.ly or Virb. But not Square Space--those assholes can go to hell.

Anyway, code documenation will be maintained on the GitHub Wiki. As an experiment, I am going to leave the edit setting on all, so don't be a Deuceberg and troll or I'll have to revert & restrict editing to just cool people.

Licensing

I will only edit & archive documentation that is licensed permissibly--or at least permissibly enough (some older software developers have funny licenses that hover in a gray area I'm not lawyer enough to understand).

If some documentation isn't explicitly licensed for at least non-commercial copy & revision, I will make a good faith effort to obtain such a license from the copyright holder. If I can't get ahold of the copyright holder, then I will consider the work orphaned and license the work as open source in the name of the copyright holder.

If the copyright holder asks me to remove the work--regardless of the reason--I will comply, but I will also make an effort to create an original open source documentation to replace it with.

That said, all documentation here is licensed exactly the same as it is licensed by the copyright holder. This copyright will be displayed in a prominent place in each document's Wiki.

What I am copyright holder to, though, is licensed broadly under an MIT license.
